# Self-hosted fonts

Fraunces and Hanken Grotesk are self-hosted so application builds do not depend on Google Fonts network access.
The optimized variable WOFF2 subsets preserve the app's Latin, Latin Extended, Vietnamese, and Hanken Grotesk Cyrillic Extended coverage.
The source families come from [Google Fonts](https://github.com/google/fonts/tree/main/ofl).
See the adjacent OFL files for redistribution terms.
